Gestionnaire de Tâches

Un système de gestion de tâches en C++ utilisant JSON pour la persistance des données.

Introduction

Ce projet est un gestionnaire de tâches développé en C++ qui permet de créer, lire, mettre à jour, supprimer et gérer des tâches stockées dans un fichier JSON.

Interface CLI

Interface en ligne de commande simple et intuitive avec affichage coloré.

Persistance JSON

Stockage des données dans un format JSON standard et lisible.

CRUD Complet

Toutes les opérations de base : Create, Read, Update, Delete.

Gestion d'État

Suivi du statut de complétion de chaque tâche.

Installation

Prérequis

  • Compilateur C++ (g++, clang++)
  • Bibliothèque nlohmann/json
  • Standard C++11 ou supérieur

Installation de nlohmann/json

# Ubuntu/Debian
sudo apt-get install nlohmann-json3-dev

# macOS
brew install nlohmann-json

# Ou télécharger depuis GitHub
# https://github.com/nlohmann/json

Compilation

Compilez le programme avec les commandes suivantes :

g++ -std=c++11 main.cpp -o todo

# Avec optimisations
g++ -std=c++11 -O2 main.cpp -o todo

Note : Le fichier output.json sera créé automatiquement s'il n'existe pas.

create

Crée une nouvelle tâche avec un titre spécifié.

Syntaxe

./todo create <titre>

Paramètres

Paramètre Type Description
titre string Le titre de la tâche

Exemple

./todo create "Acheter du pain"

# Sortie:
✓ Task created successfully!
  ID: 1
  Title: Acheter du pain

Fonction

void Create(const Item &task, json &j)

Ajoute une nouvelle tâche au tableau JSON et incrémente le compteur.

read

Affiche toutes les tâches stockées dans le fichier JSON.

Syntaxe

./todo read

Exemple

./todo read

# Sortie:
═══════════════════════════
    JSON File Content
═══════════════════════════

[
    {
        "Id": 1,
        "Title": "Acheter du pain",
        "Completed": false
    }
]

Fonction

void Read(json &j)

Affiche le contenu du fichier JSON avec une indentation de 4 espaces.

update

Met à jour le titre d'une tâche existante.

Syntaxe

./todo update <id> <nouveau_titre>

Paramètres

Paramètre Type Description
id int L'identifiant de la tâche
nouveau_titre string Le nouveau titre

Exemple

./todo update 1 "Acheter du pain et du lait"

# Sortie:
✓ Task with ID 1 was updated successfully
  New content: Acheter du pain et du lait

Fonction

void Update(json &j, int id, string content)

Parcourt le tableau JSON et met à jour le titre de la tâche correspondante.

delete

Supprime une tâche du système.

Syntaxe

./todo delete <id>

Paramètres

Paramètre Type Description
id int L'identifiant de la tâche à supprimer

Exemple

./todo delete 1

# Sortie:
✓ Task with ID 1 was deleted successfully

Fonction

void Delete(json &j, int id)

Utilise un itérateur pour supprimer l'élément correspondant à l'ID.

check

Bascule le statut de complétion d'une tâche.

Syntaxe

./todo check <id>

Paramètres

Paramètre Type Description
id int L'identifiant de la tâche

Exemple

# Marquer comme complétée
./todo check 1
✓ Task with ID 1 marked as Completed

# Marquer comme non complétée
./todo check 1
✓ Task with ID 1 marked as Not Completed

Fonction

void check(json &j, int id)

Inverse la valeur booléenne du champ "Completed".

search

Recherche et affiche les détails d'une tâche spécifique.

Syntaxe

./todo search <id>

Paramètres

Paramètre Type Description
id int L'identifiant de la tâche à rechercher

Exemple

./todo search 1

# Sortie:
═══ Task Details ═══
ID: 1
Title: Acheter du pain
Status: ○ Not Completed
═══════════════════

Fonction

void Search(json &j, int id)

Parcourt le tableau JSON et affiche les informations de la tâche trouvée.

Structure du Projet

Structure Item

struct Item {
    int id;
    string title;
    bool completed;
};

Représente une tâche avec trois attributs : identifiant, titre et statut de complétion.

Format JSON

Les données sont stockées dans output.json :

[
    {
        "Id": 1,
        "Title": "Titre de la tâche",
        "Completed": false
    },
    {
        "Id": 2,
        "Title": "Autre tâche",
        "Completed": true
    }
]

Fonctions Principales

Fonction Description Paramètres
getLastItemID() Récupère l'ID de la dernière tâche const json &j
Create() Crée une nouvelle tâche const Item &task, json &j
