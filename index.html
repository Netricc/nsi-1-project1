1. Introduction

1.1 Description générale

Cette application est un gestionnaire de tâches en ligne de commande écrit en C++. Elle lit et écrit un fichier JSON (output.json) qui contient une liste d'éléments (Id, Title, Completed). L'utilisateur interagit via des commandes : create, read, update, delete, check, search.

1.2 Caractéristiques principales

Fonctionnalité Description
Création de tâche Ajoute une tâche au fichier JSON avec un identifiant auto-incrémenté.
Lecture Affiche le contenu JSON formaté.
Mise à jour Met à jour le titre d'une tâche par ID.
Suppression Supprime une tâche par ID.
Toggle completed Bascule l'état Completed d'une tâche.
Recherche Affiche les détails d'une tâche donnée son ID.

1.3 Technologies utilisées

Composant Détails
Langage C++ (GNU g++)
Bibliothèque JSON nlohmann/json
Fichier de stockage output.json

13. Système de stockage

13.1 Fichier de données

Le fichier principal est output.json. Il contient un tableau JSON d'objets tâches.

13.2 Lecture au démarrage

Au démarrage, le programme tente d'ouvrir output.json en lecture. Si le fichier existe et contient un tableau JSON, il est parsé. Sinon, le programme initialise un tableau JSON vide.

13.3 Écriture après chaque opération

Après l'exécution d'une commande modifiant l'état (create, update, delete, check), le programme réécrit l'intégralité de l'objet JSON dans output.json en format indenté (dump(4)).

13.4 Gestion des erreurs

Si le fichier n'existe pas au démarrage, le programme crée en mémoire un tableau JSON vide et écrira le fichier lors de la première opération. Si l'ouverture en écriture échoue (permissions), une erreur est affichée et le programme renvoie code d'erreur 1.

14. Exemples d'utilisation

14.1 Workflow complet

```

# 1. Compiler

g++ -std=c++11 main.cpp -o todo

# 2. Créer deux tâches

./todo create "Acheter du pain"
./todo create "Appeler le dentiste"

# 3. Lister tout

./todo read

# 4. Rechercher un élément

./todo search 1

# 5. Mettre à jour une tâche

./todo update 1 "Acheter du pain complet"

# 6. Bascule completed

./todo check 1

# 7. Supprimer

./todo delete 2

# 8. Vérifier le JSON final

./todo read
```
